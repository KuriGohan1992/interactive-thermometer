What is Temperature?

Temperature measures how hot or cold something is. It tells us how fast the particles inside a substance are moving, the hotter it is, the faster they move. Every temperature scale measures the same thing, but uses different starting points and unit sizes.

The Interactive Thermometer

This interactive thermometer lets you visualize temperature changes in real time. Adjust the values, switch units, and see how different temperature affects the world around us.

How to adjust the values:
  • Click the thermometer at the level you want to set it to
  • Drag the "mercury" inside the thermometer
  • Press the up/down arrow keys to increase/decrease the value
  • Manually input a value by typing it on the input field
  • Click the +/– buttons beside the input field

Key Reference Points

  • Absolute zero: \(0\,\text{K} = -273.15\,^\circ\text{C} = -459.67\,^\circ\text{F}\)
  • Water freezes: \(273.15\,\text{K} = 0\,^\circ\text{C} = 32\,^\circ\text{F}\)
  • Water boils (at 1 atm): \(373.15\,\text{K} = 100\,^\circ\text{C} = 212\,^\circ\text{F}\)

Conversion Formulas

  • Celsius → Fahrenheit: \[ ^\circ\text{F} = \frac{9}{5}\,^\circ\text{C} + 32 \]
    • Fahrenheit → Celsius: \[ ^\circ\text{C} = \frac{5}{9}\,\bigl( ^\circ\text{F} - 32 \bigr) \]
  • Celsius → Kelvin: \[ \text{K = } ^\circ\text{C} + 273.15 \]
  • Kelvin → Celsius: \[ ^\circ\text{C} = \text{K} - 273.15 \]
  • Fahrenheit → Kelvin: \[ \text{K} = \frac{5}{9}\,\bigl( ^\circ\text{F} - 32 \bigr) + 273.15 \]
  • Kelvin → Fahrenheit: \[ ^\circ\text{F} = \frac{9}{5}\,(\text{K} - 273.15) + 32 \]

Quick Examples

Example 1: Convert \(25\,^\circ\text{C}\) to °F and K

\[ ^\circ\text{F} = \frac{9}{5}\cdot 25 + 32 = 77^\circ\text{F} \quad\text{and}\quad \text{K} = 25 + 273.15 = 298.15\,\text{K} \]

Example 2: Convert \(68\,^\circ\text{F}\) to °C

\[ ^\circ\text{C} = \frac{5}{9}\,(68 - 32) = \frac{5}{9}\cdot 36 = 20^\circ\text{C} \]

Example 3: Convert \(300\,\text{K}\) to °C and °F

\[ ^\circ\text{C} = 300 - 273.15 = 26.85^\circ\text{C},\quad ^\circ\text{F} = \frac{9}{5}\cdot 26.85 + 32 \approx 80.33^\circ\text{F} \]

Footnote

  • Kelvin never uses the degree symbol (write K not ˚K).
  • For real-world values, round to sensible precision (e.g., one or two decimals).
  • Negative temperatures are valid in °C and °F, but not in Kelvin.
